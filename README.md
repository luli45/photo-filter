# Web Photo Filter App

A modern, responsive web application for applying various photo filters in real-time. Built with HTML5, CSS3, and vanilla JavaScript.
Link: https://photo-app2.netlify.app/
## Features

- **Multiple Filters**: Apply various filters including Blur, Sharpen, Grayscale, Invert, Sepia, and Edge Detection
- **Real-time Preview**: See changes instantly with side-by-side comparison
- **Webcam Support**: Capture photos directly from your webcam
- **Adjustable Intensity**: Fine-tune filter strength
- **Download**: Save your filtered images
- **Fully Client-side**: No server required - works entirely in your browser

## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/photo-filter-app.git
   cd photo-filter-app/web-app
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python's built-in server
   python3 -m http.server 8000
   ```
   Then open `http://localhost:8000` in your browser.

## Deployment

### Netlify
1. Push your code to GitHub
2. Go to [Netlify](https://app.netlify.com/)
3. Click "New site from Git"
4. Select your repository
5. Set the publish directory to `web-app`
6. Click "Deploy site"

### GitHub Pages
1. Push your code to the `gh-pages` branch
2. Go to repository Settings > Pages
3. Set source to `gh-pages` branch and `/web-app` directory
4. Click "Save"
